# Example Currentness

The examples in this directory are filled examples, not automatically certified representative cases.

An example becomes a certified representative case only after it is reused in a real transcript or end-to-end project sample and still teaches the current Complex vocabulary without causing overfitting.

## Currentness Checklist

Each example should state:

- project type;
- thread or phase goal;
- beat objective;
- target function;
- linked mechanisms;
- forward artifact;
- guard or review condition;
- next route;
- what it does not prove;
- when not to copy it.

## Upgrade Rule

Upgrade an example to a certified representative case only when:

- it uses current terms from `protocol/core.md`;
- it links to at least one mechanism in `docs/mechanism-maturity.json`;
- it has a behavior case or transcript review record;
- it has been reused in a real project or high-quality redacted reproduction;
- a human review says it reduced confusion or correction cost.
